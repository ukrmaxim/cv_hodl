Некоторые мои проекты

MatisPay

Разработанно на: Ruby 3.1.2, Rails 6.1.6

Платежный агрегатор – система полноценного приема и осуществления интернет-платежей. Система имеет несколько модулей, а именно: Backend API Server – главный модуль системы. Этот модуль имеет базу данных с шифрованием всех конфиденциальных данных. Связь всех модулей системы с ним происходит только через API запросы в формате json. Client Frontend Server – личный кабинет клиента системы. Содержит информацию по проектам, кошелькам, платежам, выплатам, способам оплат и выплат, тикетам и диспутам. Admin Frontend Server – личный кабинет администратора системы. Создание и управление клиентами, проектами, кошельками, методами оплат и выплат, диспутами, тикетами, сообщениями. Processing Server - модуль обработки платежей и выплат. Имеет прямую связь с API банка или другой платежной системы. Имеет отдельную базу данных с шифрованием всех конфиденциальных данных. Payments API Server - модуль, отвечающий за создание выплат и оплат. Также содержит форму оплаты и страницы успешного и неуспешного платежа. Обмен информацией с Processing Server только через API запросы в формате json. Обмен информацией между модулями происходит только с доверенных IP.
В данном проекте я принимаю участие в проектировании системы в целом и разрабатываю backend для Admin Frontend Server и Client Frontend Server и frontend для этих модулей с использованием шаблона Bootstrap. Данная система еще находится в стадии разработки.

           

Team Space

Разработанно на: Ruby 3.1.1, Rails 6.1.4

CRM система для автоматизации бизнеса и работы компании. Система имеет двухфакторную аутентификацию с отправкой OTP кода Telegram ботом в Telegram группу. Благодаря ролям разделяется доступ пользователей к тем или иным разделам системы. В системе возможно создавать напоминания о платежах, или просто напоминания о любых задачах или встречах, которые будут отправляться в группу Telegram. В систему вносятся клиенты и их проекты. Возможно вести минимальный бухгалтерский учет движения средств по счетам компании, добавляя финансовые операции, или выставляя клиентам счета для оплаты. Все чувствительные и критические данные в базе шифруются на уровне приложения. Ко многим записям системы можно прикрепить файлы.
В будущем планируется доработка функционала получение заявок с форм с сайтов компании, разработка сборщика электронной почты, отправка электронной почты из системы и при необходимости другого функционала системы для большей автоматизации работы сотрудников. Данная система полностью спроектирована и разработана мной.

                  

World of events

Разработанно на: Ruby 3.0.0, Rails 6.1.3

Социальная сеть, в которой пользователи могут создавать события, подписываться на них, оставлять комментарии к событиям, а также добавлять к ним фотографии. Доступно ограничение доступа к событию по пин-коду. Для просмотра оригинального размера фотографии, они отображаются в виде галереи. Адрес события отображается на встроенной карте. Пользователи получают информацию об обновлении событий (добавили комментарий, фотографию или новый подписчик на событие) по электронной почте. В приложении доступна регистрация и аутентификация, используя e-mail или Github. В production среде, для хранения фотографий, добавленных к событию, используется сервис Amazon S3. Данный проект разрабатывался мной во время прохождения курса.

                 

Ask Me

Разработанно на: Ruby 3.1.2, Rails 6.1.3

Социальная сеть с возможностью задавать и отвечать на вопросы. Доступна регистрация по e-mail. Вопросы от анонимного пользователя также возможны. Для аутентификации пользователей используется самостоятельно разработанный механизм. Пользователь может настраивать свой профиль, изменяя фон, аватарку, пароль. Защита от спама и ботов орагнизованна с помощью recaptcha v.3 Для упрощения работы пользователей с приложением используется сервис хештегов. Данный проект разрабатывался мной во время прохождения курса.
